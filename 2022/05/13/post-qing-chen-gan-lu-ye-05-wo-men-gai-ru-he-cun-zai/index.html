耶5 | 胜过愚昧与狂妄，真知信仰的价值

一、去哪里寻找一个义人(vv. 1-6)

……看看有一人行公义、求诚实没有？若有，我就赦免这城。

亚伯拉罕“讨价还价”：10个人

耶利米：1个人

1、义人的沉重

思考：每一天晚上睡觉前，你会如何评价这一天？

我们常常用感觉“好”或“不好”，事情“做成”或“没做成”来评价自己的生活，然而，

⼈⽣所⾏所获的⼀切，最后只有⼀个衡量的标准：公义与邪恶

义人的沉重之在于，上帝往往透过一个义人、一个愿意顺服的人来拯救一个家族、民族、国家。

2、愚昧的、不认识神的——无知

e.g. 2.24 俄乌战争的发动到它后来的发展轨迹，让全世界都震惊

如何解决对付愚昧与无知：

  • 要知道自己不知道什么：世界、自己、他人、上帝？

  • 不断地学习、成长，真正认知自己需要认知的

  • 找到可以信任的，真正有认知的人，多听

3、尊大、受过教育的，齐心抛弃神的束缚——狂傲

有一些真正的认知，但是因着这种知识而变得狂傲，自以为可以利用这一切，达到自己的目的，而不是那创造主，主宰者

真理不是达成我们目的工具，而是我们需顺服的对象

基督教必须知道，它已经不是医治世界痛苦的万灵药的专卖者，相反地，上帝已经把人类得救的钥匙，从它手中夺去，给了别人。总之一句话，基督教必须大彻大悟，让旧的躯壳死去，让新的生命来临。
——吴耀宗“基督教的改造”载《天风》第173 期（1949 年7 月30 日）

二、如何能赦免？（vv. 7-13）

1、行淫的犹大

2、诡诈攻击神

3、“先知的话必成为风，道也不在他们里面”

三、外邦的攻击(vv. 14-19)

1、耶和华的话如烈⽕，是真正的审判

2、毁灭与被掳是你们的命运，既然你愿意

认为“信仰是解决温饱之后才需要考虑的问题”的人，将一辈子为温饱而奔波忙碌

你把什么放在信仰之后，就会一生侍奉它

四、愚蠢的作恶者，你以为这会如何收场？（vv. 20–31）

1、罪的本质是⼀种愚昧⽆知

2、不知什么叫“惧怕”，没有敬畏

3、空有财富、权势，却不运用这⼀切来行公义、好怜悯

  • 中国战国时期贵族精神的没落与消失

    富人比穷人仅仅是多有钱与权，思考的本质仍是“如何混口饭吃”，只不过是吃得更好而已

4、“到了结局你们怎样⾏呢？”

五、要真正认识基督信仰

1、对信仰的错误认知

大部分人可能都严重低估了信仰的意义与可能性

很多的人刚刚瞥见了基督的荣耀，没有看到那个基督⾝体真正的意义

  • 当你把信仰仅仅当作人生的一部分时，你就很难看清楚圣经到底在讲什么；
  • 当你仅仅聚焦信仰对你的⼈⽣有什么影响，不能允许信仰改变你对整个⼈⽣的看法时，你就把你的信仰禁锢在⼀个极其狭隘的领域；
  • 当你仅仅把教会当作传教的⼀个⼯具，⾥⾯的⼏个⼈仅仅是为了讨⽣活的时候，你就不会明⽩，什么叫做“建立上帝的国度”。

2、两种秩序

这个世界，是偶像秩序与耶和华秩序斗争，这两种秩序都会成型在⼀群具体的人身上，这群人彼此间的互动、与外界的互动，表现出这个秩序的本质。

个体化的信仰，仅仅能构建一个人在头脑里理解、想象的秩序，或信心中确信的秩序，而我们需要的，是真正人与人之间道成肉身的秩序。

建⽴这样的秩序不容易，但是却是最有价值的事情。

3、信仰的意义与可能性

  • 从这一生来看，信仰引导你走向真理、恩典、爱的生命模式，给其他的人最大的服事与祝福
  • 从你对其他人的影响来看，你如同人粒麦子死了，埋在土里，成为其他人生命的营养，不断地传承下去
  • 从永恒来看，⼀切你对世界、自己、他人的影响都会堕入虚空，唯有在永恒信仰中的行动，才会真正的意义

思考：⽣命只有⼀次，你要怎么过这⼀⽣？

今天的回放👇

回过头再咂摸王牧分享的最后一部分时，我耳边蓦然响起一首歌——《存在》：

多少人走著却困在原地 / 多少人活著却如同死去 / 多少人爱著却好似分离 / 多少人笑著却满含泪滴

谁知道我们该去向何处 / 谁明白生命已变为何物 / 是否找个藉口继续苟活 / 或是展翅高飞保持愤怒

我该如何存在

多少次荣耀却感觉屈辱 / 多少次狂喜却倍受痛楚 / 多少次幸福却心如刀绞 / 多少次灿烂却失魂落魄

谁知道我们该梦归何处 / 谁明白尊严已沦为何物 / 是否找个理由随波逐流 / 或是勇敢前行挣脱牢笼

我该如何存在

小时候听这歌，单纯是因为喜欢它的旋律，其实并不完全理解歌词。那时候的我还没有诗和远方，也怎么不谈理想和希望，所求的就是把眼前的题目解出来，应付好每一场考试。

后来，我信了主，在教会和团契里感受到温暖与爱，同时也被教导要在生活中活出信仰，多跟周围的人传福音。

传福音的时候要做见证，我以前会先讲信主前的我是如何不堪，而信主后我发生了怎样的变化，由此引出结论——信仰对人生有极大的正面影响。

难怪每个听完我分享的人都会对我说：“有信仰挺好的。”

我想，他们没有说出来的另一句话可能是“不过，我暂时不需要用到（宗教）信仰（这个工具）”。

其实，我很能理解他们的想法，毕竟我也曾是不信的一员。在相当长的一段时间里，信仰对我而言就只是生活的一部分罢了。

记得大学四年的周末和假期，我几乎全泡在教会和团契里。别人问我团契是什么意思的时候，我会跟对方解释道：团契就和社团差不多，一群人有一个目标一起做事，只是团契里的人联系更紧密，像一家人。我偶尔也会跟着一起去传福音，但更多的是把它当作一种任务，而不是视之为我的人生使命。

时至今日，我仍然未能完全认识到信仰的意义与可能性，尽管学了不少课程，但是很多东西还停留在理论层面，我没能活出来。

然而，我也有成长：对罪越来越敏感，也更加敢于向神和人剖白自我。

因着自己总是一次又一次地跌落罪的泥淖中，我会忧伤痛悔，可是我却不再感到绝望，因我知天父一直在陪着我，祂也必扶持我重新站立。
